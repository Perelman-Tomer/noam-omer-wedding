# נועם ועומר - הזמנה לחתונה

אתר אינטרנט לחתונת נועם ועומר - דף אחד עם גלילה הכולל סרטון, הזמנה, ספירה לאחור, וקישורים לפעולות.

## 🚀 התחלת עבודה

### התקנה
```bash
npm install
```

### הרצת מצב פיתוח
```bash
npm run dev
```
האתר יהיה זמין בכתובת: http://localhost:3000

### בניית גרסת ייצור
```bash
npm run build
```

### בדיקת קוד
```bash
npm run lint
```

## 📁 מבנה הפרויקט

```
src/
├── components/
│   ├── VideoSection.jsx      # סרטון המעטפה עם fadeout
│   ├── InvitationSection.jsx # הזמנה לחתונה עם רקעים שכבתיים
│   ├── CountdownTimer.jsx    # ספירה לאחור ל-16.06.2026
│   └── ActionButtons.jsx     # כפתורי פעולה (ווייז, יומן, גוגל דרייב)
├── styles/
│   └── global.css           # סגנונות גלובליים + גופנים עבריים
├── assets/
│   ├── fonts/               # גופנים עבריים
│   ├── images/              # תמונות רקע ולוגו
│   └── video/               # סרטון המעטפה
├── App.jsx                  # קומפוננט ראשי
└── main.jsx                 # נקודת כניסה

public/
└── assets/                  # נכסים סטטיים
```

## 🎬 סרטון הפתיחה
- **סרטון מלא מסך** - האתר מתחיל עם סרטון מעטפה נפתחת (ללא אפשרות גלילה)
- **Fadeout חלק** - אחרי סרטון, fadeout הדרגתי של הוידאו במשך 1.5 שניות
- **מניעת גלילה** - במהלך הסרטון והמעבר הגלילה חסומה
